Cloud Learning at Global Oracle OpenWorld 2020

Elisa Roberts
Global Marketing Director

As 2020 begins, we’re offering our readers a complete guide to Cloud Learning sessions and activities at Oracle OpenWorld events around the globe this year.   

The Oracle University Cloud Learning Team will have a kiosk and speaking sessions at each event. How can you leverage Oracle OpenWorld to accelerate your cloud adoption strategy?  

  • Register for our sessions to learn about the latest innovations in cloud learning
  • Stop by our kiosk to get a live demo of Oracle Guided Learning and find out how much time and expense it can save your organization
  • Ask our learning specialist about Cloud Learning Subscriptions, Oracle Launchpad, and Oracle Certification

 

Interested in learning about solutions specific to your needs? Contact Oded Ilan at oded.i.ilan@oracle.com to set up a personalized meeting at the event.

Looking forward to seeing you at:

Register for Oracle OpenWorld Middle East 2020

Sign up for our 3 Lightning Sessions (15 minutes each):

Lightning Sessions:

  • LHT1844 - The New Oracle Guided Learning: A Revolution in Oracle SaaS Adoption – Speaker, Wim de Haas, Oracle University
  • LHT1845 - Improve the User Experience in Oracle ERP Cloud with Oracle Guided Learning – Speaker, Franck Kervella, Oracle University
  • LHT1846 - Learning Across the Cloud Lifecycle: Oracle Guided Learning and More – Speaker, Aon Siddiqui, Oracle University

Visit the Oracle University kiosk in Zone 4 to learn more about cloud learning options and get an Oracle Guided Learning demo.
